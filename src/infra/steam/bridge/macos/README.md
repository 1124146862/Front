# macOS bridge files

Place the macOS Steam runtime files here when testing on macOS:

- `libsteam_api.dylib`
- `steam_appid.txt`
- `steam_id_bridge.exe` built from `SteamIdBridge.cs`

Suggested runtime layout on macOS:

- `SteamIDGet/macos/libsteam_api.dylib`
- `SteamIDGet/macos/steam_appid.txt`
- `SteamIDGet/macos/steam_id_bridge.exe`

If you move the bridge outside this prototype folder later, keep these three files together.
